## **1. 스크럼 요약**

* **전체 진척도:** 백엔드 시작
* **금일 목표:** 프로젝트 구조 정하기



## **2. 주요 논의 안건**

**안건 1: 프로젝트 구조 정하기**

* **내용:** 도메인 구조 정하기
* **의견:** 관리자와 직원을 멤버 폴더에 넣을지 말지

**안건 2: 개인별 진행 사항**

* **내용:** 현재 어떤 것이 어디까지 진행되었는지, 앞으로 어떤 걸 할것인지 공유

**안건 2: 코드 리뷰**

* **내용:** 언제 코드 리뷰를 할 것인지

## **3. 결정 사항**

* **프로젝트 구조 정하기: [프로젝트 메인 구조(백엔드)](https://projfinalhq.slack.com/docs/T0A7DKPPL0H/F0A753DP329)**
* **개인별 진행 사항**
    * 원석: 로그인/회원가입 컴포넌트 분리(코드 학습 중), 백엔드 세팅함
    * 승원: 관리자 페이지 기프티콘 끝 직원 페이지 기프티콘 할 예정, 멤버 더미데이터 만들어서 연동할 예정
    * 정훈: ERD, 엔티티 끝 근태 페이지 코드 학습 중(10% 완료) 더미데이터 만들어야함
    * 치호: 부서정보/상담신청 프론프엔드 끝 코드 학습 중 백엔드 로직 구현해야함
    * 준언: 직원 마이페이지 백엔드 구현 중 연동 해야함(select는 되는데 update가 안 됨)(30% 진행)관리자 마이페이지도 구현해야함
* **코드 리뷰**: 기능을 먼저 완료한 후 코드리뷰하기로함
