## 2. 🚧 직면한 문제 & 어려움

### 1) 렌더링 오류 (White Screen)
- **문제:** 앱 실행 시 화면이 하얗게 나오고 아무것도 뜨지 않음.
- **원인:**
    1. 파일 경로의 대소문자 불일치 (`timer` vs `Timer`, `Routes` vs `routes`).
    2. 컴포넌트 import 이름과 실제 함수명 불일치 (`HOME` vs `Home`).
- **해결:** 파일명과 import 경로의 대소문자를 정확히 일치시키고, 컴포넌트 export 방식을 통일하여 해결.

### 2) 레이아웃 흔들림 현상
- **문제:** 홈 화면(스크롤 있음)과 다른 페이지(스크롤 없음) 이동 시 헤더가 미세하게 움직임.
- **원인:** 스크롤바가 생기면서 화면 너비(Width)가 달라지기 때문.
- **해결:** `index.css`에 `html { scrollbar-gutter: stable; }` 속성을 추가하여 스크롤바 영역을 미리 확보함.

### 3) Git 메인 브랜치 실수
- **문제:** 작업 내용을 실수로 `main` 브랜치에 직접 커밋하고 푸시함.
- **해결:**
    1. 현재 코드를 `feat/layout` 브랜치로 피신.
    2. `git reset --hard` 명령어로 로컬 `main`을 작업 전 시점으로 되돌림.
    3. `git push --force`로 원격 저장소 `main`을 초기화하여 복구 완료.
