## 목표

- React의 주요 개념(Hooks, Props, Routing 등)과 학습한 라이브러리들을 활용하여 자유 주제로 완성도 있는 **SPA**을 구현하는 것이 목표입니다.

## 필수 구현 기능

1. **게시판형 CRUD 기능**
    - 게시글 등록, 목록, 수정, 삭제
    - 게시글 상세 페이지
    - 리스트 정렬/검색 (선택 사항)
2. **로그인/회원가입 CRUD 기능**
    - 로그인/로그아웃 처리
    - 회원가입 시 유효성 검사
    - 로그인 후 마이페이지 or 유저 정보 수정 기능
3. **페이지 분리 및 라우팅**
    - 최소 3개 이상의 라우트
    - `react-router-dom`을 활용한 라우트 구성
    - 존재하지 않는 경로 처리(404 페이지)
