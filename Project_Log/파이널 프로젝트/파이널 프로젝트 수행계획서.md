# 2026년 KDT 파이널 프로젝트 수행계획서

## 1. 프로젝트 개요

### 가. 프로젝트 정보
| 구분 | 내용 |
| :--- | :--- |
| **프로젝트명** | Calm Desk |
| **주제** | 콜센터 직원을 위한 HR SaaS 플랫폼 |
| **기술분야** | SW (웹 애플리케이션) |
| **수행기간** | 2026. 01. 06. ~ 2026. 02. 27. |
| **성과목표** | 감정 노동자 보호법을 준수하는 실무형 HR SaaS 플랫폼 설계 및 구현<br>클라우드 환경(AWS)에서의 배포 및 운영을 통한 풀스택 개발 경험 축적 |

### 나. 프로젝트 소개 및 제안 배경
* **목적:** 콜센터 상담 인력이 겪는 고강도의 감정 노동과 정신적 부담을 체계적으로 관리하고 보호할 수 있는 HR·건강 통합 관리 시스템 구축.
* **기존의 한계:** 기존 HR 시스템은 행정적 편의성(출퇴근, 스케줄링)에만 초점이 맞춰져 있어, 감정 소모나 정서적 리스크를 관리하는 기능이 부재함.
* **제안 내용:** 상담사 보호 중심의 데이터 기반 HR 및 멘탈케어 SaaS를 제안하여 생산성 향상과 인권 보호 동시 달성.

### 다. 주요 기능 요약
1. **감정 노동 케어 & 쿨다운(Cool-down):** 악성 민원 발생 시 '긴급 보호 휴식' 신청 및 관리자 알림, 감정 상태 기록 및 위험군 식별.
2. **유연 근무 & 워케이션:** 근태 인정 프로세스, 스트레스 지수와 연동된 힐링 프로그램 및 상담 연계.
3. **직원 관리 및 모니터링:** 실시간 근무 상태(통화중, 대기, 이석 등) 대시보드 시각화.
4. **스트레스 통계:** 기분 상태와 쿨다운 횟수를 합산해 개인별 스트레스 지수 산출 및 통계화.

## 2. 프로젝트 수행계획

### 가. 주요 기능 명세

| 구분 | 기능 | 설명 |
| :--- | :--- | :--- |
| **S/W** | **직원 관리** | 직원의 정보(등록/조회/수정)와 부서/팀 단위 관리, 근태 관리 기능 |
| **S/W** | **스트레스 리스크 분석** | 누적된 쿨다운 로그와 감정 상태 데이터를 정량적으로 분석<br>스트레스 지수에 따른 위험군 자동 식별 |
| **S/W** | **업무(스케줄) 분석** | 직원들의 근태에 따른 업무 인원 파악<br>직원의 쿨다운, 자리비움, 근무 중인 인원 파악 가능 |
| **S/W** | **맞춤형 치유 솔루션** | 고위험군 대상 강조 표시 및 결재, 근태 반영 |
