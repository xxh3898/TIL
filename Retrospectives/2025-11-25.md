## 📚 학습 내용
> 오늘 공부하면서 새롭게 배운 내용이나 정리한 개념

- **컴포넌트 구조화 (Component Hierarchy)**
    - 모든 코드를 `App.jsx`에 작성하지 않고, 기능별로 컴포넌트를 쪼개어 관리하는 법을 익혔다.
    - 구조: `App` (데이터 관리) -> `Videos` (리스트 레이아웃) -> `VideoCard` (개별 아이템 UI)

- **Props 심화 & 데이터 전달**
    - 배열 데이터를 부모 컴포넌트에서 자식 컴포넌트로 전달하는 과정을 실습했다.
    - `App`에서 정의한 `videoList` 배열을 `Videos` 컴포넌트로 보내고, 거기서 다시 `map()`을 돌며 `VideoCard`로 개별 객체를 전달했다.

- **Styled-Components 파일 분리**
    - 컴포넌트 파일(`jsx`) 안에 스타일 정의가 섞여 있으면 코드가 너무 길어진다.
    - `VideoCard.styled.js` 처럼 스타일만 전담하는 파일을 따로 만들고, `export const`를 사용하여 스타일드 컴포넌트를 내보낸 뒤 `import`해서 사용하는 패턴을 적용했다.

## 💣 Trouble Shooting
> 발생한 문제와 해결책

- **문제**: 스타일 파일을 분리했는데 컴포넌트에서 적용되지 않고 에러가 발생함.
- **원인**: `VideoCard.styled.js`에서는 `export const` (Named Export)를 사용했는데, 받을 때 `import Container from ...` (Default Import) 처럼 받으려 해서 찾을 수 없다는 에러가 뜸.
- **해결**: `import { Container } from ...` 처럼 중괄호(`{}`)를 사용하여 Named Import 방식으로 수정하여 해결함.
