## 📚 학습 내용
> 오늘 공부하면서 새롭게 배운 내용이나 정리한 개념

- **JSX (JavaScript XML)**
    - 자바스크립트 확장 문법으로, JS 내부에서 HTML과 유사한 코드를 작성할 수 있게 해준다.
    - `{}`(중괄호)를 사용하여 변수나 표현식을 포함할 수 있다.
    - **조건부 렌더링**: 삼항 연산자(`? :`)나 AND 연산자(`&&`), OR 연산자(`||`)를 활용하여 조건에 따라 다른 UI를 보여줄 수 있다.

- **React에서의 스타일링 방법**
    1. **Inline Style**: 객체 형태로 작성하며, `camelCase` 속성명을 사용한다. 재사용성이 낮아 권장되지 않는다.
    2. **External CSS**: 별도 CSS 파일을 `import` 하여 사용한다. (`className` 속성 사용)
    3. **Styled-Components**: CSS-in-JS 라이브러리로, 컴포넌트 레벨에서 스타일을 정의한다.
        - JS 파일 내에서 CSS 문법을 그대로 사용할 수 있다.
        - 클래스명 충돌 문제를 방지하고, `props`를 활용한 동적 스타일링이 가능하다.

- **Props & List Rendering**
    - **Props**: 부모 컴포넌트에서 자식 컴포넌트로 데이터를 전달할 때 사용하는 객체.
    - **Map()**: 배열 데이터를 순회하며 컴포넌트 리스트를 렌더링할 때 사용한다. 이때 고유한 `key` props가 필수적이다.

## 🧑🏻‍💻 코드 예제
> 수업 시간에 작성한 핵심 코드나 직접 변형한 코드

```jsx
// 1. Styled-components 활용 예제
import styled from 'styled-components';

// 스타일이 적용된 div 컴포넌트 생성
const RoundBox = styled.div`
    position: absolute;
    top: ${props => props.top || 50}px; // props를 활용한 동적 스타일링
    width: 200px;
    height: 200px;
    background-color: blue;
    border-radius: 50px;
    color: white;
`;
```

```jsx
// 2. JSX 조건부 렌더링 & 리스트 출력
const JavaScript = () => {
  const num = 100;
  const userList = [{ name: "최지원" }, { name: "김지원" }];

  return (
    <>
      {/* 삼항 연산자 */}
      <li>{num > 10 ? "10보다 큼" : "10보다 작음"}</li>
      
      {/* AND 연산자 (조건이 참일 때만 렌더링) */}
      {num !== 10 && <li>num은 10이 아님</li>}

      {/* Map을 이용한 리스트 렌더링 */}
      {userList.map((u, index) => <li key={index}>{u.name}</li>)}
    </>
  )
}
```
